# Progetto-API-24-25
Progetto in C che modella una mappa esagonale per il calcolo di percorsi a costo minimo. Supporta inizializzazione della mappa, aggiornamento dinamico dei costi su area, gestione di rotte aeree monodirezionali e query di costo minimo tramite Dijkstra con min-heap. Input gestito via comandi testuali standard
